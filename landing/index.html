Why Choose TransKit?

Built for privacy, powered by AI, designed for simplicity

Privacy-First Design

Default to Chrome's built-in Gemini Nano AI for offline translation. No tracking, no data collection, no external API calls required.

Offline-Capable

Works without internet when using Chrome's native AI. Perfect for secure environments or when traveling.

6 AI Providers

Choose from Chrome Built-in AI, Google Gemini, OpenAI, Groq (fast), Ollama (local), DeepL, or custom OpenAI-compatible endpoints. Switch providers anytime.

17+ Languages

Support for English, Vietnamese, Chinese, Japanese, Korean, Spanish, French, German, Portuguese, Italian, Russian, Arabic, and more.

100% Open Source

Fully transparent code on GitHub. Review, contribute, or fork the project. Built by the community, for the community.

Smart Auto-Detection

Automatically detects the source language when not specified. Intelligent language mapping with multi-language synonyms.
